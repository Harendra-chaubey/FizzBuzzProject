# FizzBuzz API

This is a .NET Core Web API for the FizzBuzz game. The API accepts an array of strings as input and returns the corresponding FizzBuzz results.

## How to Run

1. Clone the repository from GitHub.
2. Open the solution in Visual Studio 2022.
3. Set the API project as the startup project.
4. Press F5 to run the application.

## How to Test

1. Open the Test Explorer in Visual Studio.
2. Ensure the test project is included in the solution.
3. Run all tests to ensure they pass.

## Requirements

- .NET Core Framework
- Visual Studio 2022
- NUnit
- Moq
